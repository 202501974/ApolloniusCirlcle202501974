# 🔵 아폴로니우스 내분점 실험 앱

두 점 A와 B 사이에서 `AP : PB` 비율을 입력하면 내분점 `P`를 계산하고 시각화하는 Streamlit 앱입니다.

### 실행 방법

1. 요구사항 설치

   ```
   pip install -r requirements.txt
   ```

2. 앱 실행

   ```
   streamlit run streamlit_app.py
   ```

### 앱 기능

- 점 A와 B의 좌표 입력
- `AP : PB` 비율 입력
- 계산된 내분점 P 위치 표시
- AB 선분 위의 내분점 그래프 출력
